Orbital Larp Cannon

This plugin is a natively-ported and heavily optimized version of the Orbital Strike Cannon plugin originally by xFairyzz.
It has been completely rewritten in Kotlin to drop external dependencies and run natively on the server architecture for maximum performance.

Version Support:
- Paper 1.20.x up to 1.26.x
- Folia 1.20.x up to 1.26.x
(Automatically detects and routes to the correct native scheduler).

Note: A small amount of AI assistance was used during the porting process to refactor the schedulers and optimize chunk unloading mechanics.
